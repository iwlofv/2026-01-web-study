# 프로젝트 제출 폴더

이 폴더는 학생 제출 구조를 안내하기 위한 기준 폴더입니다.
실제 학생 제출물은 각자 fork 레포의 `projects/<github-id>/` 아래에서 관리합니다.

권장 구조:

```text
projects/
  <github-id>/
    week1/
    week2/
    final/
```

규칙:
- 폴더 이름은 본인 GitHub 아이디를 사용합니다.
- 작업은 본인이 fork한 레포의 `main` 브랜치에서 하고, 제출은 Pull Request로 진행합니다.
- 공용 레포 `main` 브랜치에는 직접 push하지 않습니다.
- `main` 브랜치에는 학생 제출 폴더를 만들지 않습니다.
- Pull Request는 제출 확인용으로 열고, 기본적으로 merge하지 않습니다.
- 다른 사람 폴더는 수정하지 않습니다.
- 발표용 결과물은 `final/` 또는 별도 안내된 폴더에 정리합니다.
